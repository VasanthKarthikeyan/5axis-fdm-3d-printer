# 5 Axis FDM 3D Printer

A research-driven design and development of a hybrid 5-axis FDM 3D printer.

## Objectives

- Develop a hybrid CoreXY–polar motion architecture
- Enable multi-axis additive manufacturing
- Build a digital twin of the printer
- Perform kinematic and dynamic modeling

## Architecture

- XYZ: CoreXY gantry
- A-axis: nozzle rotation
- Polar rotating circular bed

## Documentation

Project documentation is available in the /docs folder.

## Project Goals

- High accuracy across entire build volume
- High speed and acceleration
- Robust calibration
- Expandability to 6 DOF
